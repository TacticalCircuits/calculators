Buck / Boost Converter Rough Sizing

Ideal CCM estimates for duty cycle, inductor ripple, capacitor ripple, and input current.

Equations used
Buck: D ≈ Vout / Vin, ΔIL ≈ (Vin − Vout)D / (Lfsw) Buck: ΔVout ≈ ΔIL / (8fswC) Boost: D ≈ 1 − Vin / Vout, ΔIL ≈ VinD / (Lfsw) Boost: ΔVout ≈ IoutD / (fswC) Input current estimate: Iin ≈ (VoutIout) / (ηVin)

These are first-pass ideal CCM estimates. Check controller datasheet equations, ESR, DCR, switching losses, compensation, and saturation current before committing a design.

PCB, RF & Digital

PCB Copper Trace Resistance

Approximate DC resistance using copper thickness in oz/ft² and temperature correction.

Equations used
1 oz copper thickness ≈ 34.8 µm ρ(T) = ρ20 · [1 + α(T − 20°C)], ρ20 ≈ 1.724e−8 Ω·m, α ≈ 0.00393/°C R = ρL / A = ρL / (width · thickness) Voltage drop = I · R, Copper loss = I²R

Microstrip Impedance Approximation

Thin-conductor quasi-static estimate for outer-layer microstrip.

Equations used
u = W/h εeff ≈ (εr+1)/2 + (εr−1)/2 · [1/√(1+12/u)] with a small correction for u ≤ 1 u ≤ 1: Z0 ≈ (60/√εeff) · ln(8/u + 0.25u) u > 1: Z0 ≈ 120π / [√εeff · (u + 1.393 + 0.667ln(u + 1.444))]

Use a field solver or board-house stack-up tool for production controlled-impedance work, especially with soldermask, copper roughness, differential pairs, or thick copper.

ADC Resolution & Acquisition Time

Ideal LSB, code, quantization SNR, and source-R/sample-C settling estimate.

Equations used
Codes = 2^N, LSB = Vref / 2^N Ideal code ≈ round(Vin / LSB), clipped to 0…2^N−1 Ideal quantization SNR ≈ 6.02N + 1.76 dB τ = Rsource · Csamp Settling to 0.5 LSB: tacq ≥ (N + 1)ln(2) · τ
Assumptions: most calculators use ideal component equations unless the section notes otherwise. Validate against tolerances, parasitics, component ratings, temperature limits, manufacturer data, and safety requirements before release or energizing hardware.
